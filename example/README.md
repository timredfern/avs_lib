# AVS Library Example

This example demonstrates how to use the AVS library as a standalone application.

## Building

From this directory:

```bash
mkdir build
cd build
cmake ..
make
./avs_example
```

## Features

This example shows:
- Basic AVS renderer setup
- Effect loading and management
- Audio data processing
- Simple console-based visualization

The example creates a basic effect chain and runs a simple simulation with test audio data.
